#######
Styling
#######

.. toctree::
    :glob:
    :hidden:

    *

Styling in seamless can be done in multiple ways.

One way is to use the :ref:`StyleObject <style-object>` class.

Another way is to use :ref:`CSS Modules <css-modules>`.
More on CSS Modules can be found in the CSS Modules `GitHub <https://github.com/css-modules/css-modules>`_ documentation.
